# 实验1 C 语言编程

## 一、实验目的

1. 复习C语言编程。

## 二、实验内容
1. 熟悉C语言编程。软件调试一下题目： 
   1. 两数相加 
   2. 两数比大小 
   3. 1+2+3+……+10 
   4. 其他简单数学计算 ……… 

## 三、实验步骤 （简单操作可参见实验2中项目创建步骤） 
1. Keil软件，使用“Project”菜单中的“New μVision Project”选项来创建项目，放 置到你指定的文件夹。 
2. 选择单片机型号AT89C51 
3. 编写源程序，并保存（扩展名用C），例程见图1-1。 
4. 将文件添加到工程中。 
5. 调试。
